      Welcome to our website! Here's some useful information to help you navigate it:
      1. The Measurement menu displays the 30 most recent measurements.
      2. There are dropdown menus for Temperature and Wind Speed,
         allowing you to select data for those specific time.
      3. To access more data types, use the Selection menu.
         Each data type has its own dropdown menu.
      4. Inside each table, there is a sorting function that allows you to sort
         the data by different criteria. You can choose to sort the data in either
         increasing or decreasing order, depending on your preference.
